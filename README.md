# TYPO3 - Roadmap for improving the Quality Assurance

This talk elaborates ideas I'd like to introduce for better Quality Assurance for TYPO3.

## View the presentation

If you want to view the presentation make sure to have installed all dependencies:

	bundle install

Then you can start the server:

	bundle exec showoff serve

The presentation is available at `http://localhost:9090`.
